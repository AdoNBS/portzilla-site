# GatsbyJS - Static site boilerplate

Based on Reactjs.org source code.

Featuring:

- GatsbyJS engine
- AntD for React.js components
- Emotion for CSS-in-JS
- mdx for static content

## Development

```
yarn
yarn dev
```

or

```
npm i
npm run dev
```
